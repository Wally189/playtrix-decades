How this page fits with the console

Console = launch pad · Tools = cupboard
The Console page is your “today” view – the places you launch from every day. The Tools page is where you keep everything tidy in the background.
Over time, you can promote or demote tiles between the two. If something matters every single day, it belongs on the Console. If it’s useful but not constant, it can happily live here.
