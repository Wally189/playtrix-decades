How to switch themes
Tap the Theme button in the top-right of any page to cycle through themes.
Your choice is saved automatically, so PlayTriX always opens with your favourite decade.

Backup tools

Export setup
Save your PlayTriX configuration as a file on your device.
Import setup
Restore a saved configuration from a file.
Copy backup as text
Gets a long encoded string you can paste into Notes.
Restore from text
Paste the backup string here to rebuild your console.
Why backups matter
Browsers erase stored data when you clear history or switch devices. Backups make your PlayTriX console portable and future-proof.
No data ever leaves your device – backups belong entirely to you.

Reset

Reset PlayTriX
Clears all saved tiles, themes, favourites and settings.
You can restore from a backup afterwards.

Security

Password lock
The lock screen is not a replacement for your device password – it simply keeps your console private if someone else opens PlayTriX.
Password setup and removal lives on the dedicated Password page.
